# Changelog

All notable changes to VPN Master Panel Lite will be documented in this file.

## [1.5.0] - 2026-02-22

### Changed
- OpenVPN-only: removed WireGuard backend/service/dependencies and all WireGuard UI/docs
- Removed Docker artifacts (Dockerfile/docker-compose) to focus on direct server installation
- Simplified settings/subscription UI (removed non-OpenVPN formats and QR/URI export options)

## [1.0.0] - 2025-XX-XX

### Added
- Initial release
- Basic dashboard
- User management
- OpenVPN support
- Basic monitoring
